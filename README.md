# SnapMatch

**SnapMatch** — это система интеллектуального анализа резюме кандидатов с помощью ИИ. Она помогает быстро и точно оценить соответствие соискателя конкретной вакансии. Проект ориентирован на бизнес-сегмент, связанный с наймом и подбором персонала.
Пока это MVP, направленный на первичный отбор кандидатов на вакансии, связанные с IT.
